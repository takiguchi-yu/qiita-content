あまり需要はないが、ひとつのパソコンで複数の github アカウントを使い分ける場合は、以下のようにすると切り替えができる。

#### 1. SSH キーを切り替える

```sh:~/.ssh/config
Host github.com
  IdentityFile ~/.ssh/github_userA_keyA
  #IdentityFile ~/.ssh/github_userB_keyB
```

#### 2. メールアドレスを切り替える

```sh:~/.gitconfig
[user]
  name = userA
  email = userA@example.com
  #name = userB
  #email = userB@example.com
```
