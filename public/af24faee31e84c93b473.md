| 用語               | 説明                                                                                                                                                                       |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| レンダラープロセス | オモテの見える部<br>HTML等で記述する部分                                                                                                                                   |
| メインプロセス     | Node.js で記述する部分<br>オモテから見たときのバックエンドの部分にあたる                                                                                                   |
| ContextBridge      | レンダラープロセスから直接メインプロセスを操作させないための仕組み<br>セキュリティ向上のための仕組みで、このワンクッションを挟むことで OS にイタズラされないようにしている |
